# @tapsellorg/eslint-plugin

Tapsell custom eslint plugin

## Installation

```
npm i -D @tapsellorg/eslint-plugin eslint
```

## Usage

In your ESLint config file `.eslintrc`:

```js
module.exports = {
  extends: 'plugin:@tapsellorg/angular',
  root: true,
};
```
